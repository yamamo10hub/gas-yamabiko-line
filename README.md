### 受付可能なコマンド
ヘルプ
```
指定したメッセージ(コマンド一覧)の表示
```
曜日？
```
今日の曜日を表示する
```
ゴミの日？
```
リストに記載されている明日のゴミ分別を表示
```

リスト
```
現在のスプレッドシート上のTodoエントリの表示
```
入力？
```
Todoリストに書き込む為のフォームのリンクを表示する
```

検索？[word]
```
Todoリストの中身を指定wordで検索し、有無を返答する
```
